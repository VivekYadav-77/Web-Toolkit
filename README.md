## Getting Started
1. Clone this repository:

```bash
git clone https://github.com/VivekYadav-77/web-toolkit.git
```

2. Navigate to the project folder:

```bash
cd Web-Toolkit
```

3. Open the project in your browser:

```bash
cd bookmark
open b.html
```
or
```bash
cd studentregistry
open registry.html
```
